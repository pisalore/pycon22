## pycon22 

1. Clone this repo
   ```shell
   git clone git@github.com:pisalore/pycon22.git
   ```

2. Create a `virtualenv`


4. Install `jupyter` and `nbconvert`


5. Run notebook:
   ```shell
   cd pycon2022
   jupyter notebook
   ```
6. Convert to slides
    ```shell
   jupyter nbconvert pycon22.ipynb --to slides
   ```
